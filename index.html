Game Boy

從 20 世紀 90 年代開始，Game Boy 一直是掌上遊戲機的代表者。低廉的價格、種類繁多的遊戲、耗電量不大、加上是由當時遊戲界霸者的 Nintendo 出品，所以即使只是黑白畫面亦能打敗同期機能較強的 GameGear 和 NEC PC-GT。

甚至在十幾年後的 1998 年，SNK 的 NeoGeo Pocket 也敗在了 Game Boy 的手下。

1989
Game Boy

Nintendo 經過 100 年的經營，正式發售 Game Boy 遊戲機。

因為以長時間遊玩為主要考量，Game Boy 只採用解析度 160 × 144 的反射式灰階液晶螢幕，雖然較 ATARI Lynx、SEGA Game Gear 等同期彩色競爭機種來說顯得較不亮眼，不過最長 35 小時的續航力、12800 日圓的低廉價格，加上 Tetris 等殺手級的遊戲陣容，使 Game Boy 擊敗眾多競爭者稱霸掌機市場近 10 年。

1996
Game Boy Pocket

相比於上一代的 Game Boy 縮小了近 30% 的體積，並改良了 LCD 螢幕、更加省電，但在機能方面並沒有太大的變化，所以可以說是 Game Boy 的輕薄最佳化版。

1998
Game Boy Light

是任天堂第一款帶背光的便攜式遊戲機，鑒於液晶顯示屏的特性，以往的 Game Boy 主機只能夠在光源充足的情況下使用，限制了掌機隨時遊玩的特性，因此加入了屏幕背景燈光的功能，使得即使在漆黑的房間中遊戲的畫面也能夠顯示得很清楚。而經過改良後使得主機的耗電量下降，電池就具有了更長的使用壽命。不過由於僅比下一代的 Game Boy Color 早約 6 個月發售，再加上單在日本銷售，因此出貨量不多，是系列幾種中較為稀少的一款。

Game Boy Color

顯著的不同就是，擁有了一個最大發色數為 32000，可以同顯 56 色的 TFT 彩色液晶螢幕。現在來看 Game Boy Color 上面所登場的遊戲，其畫面、音效品質與製作水準，都已經全面超越了當年的時代。

2001
Game Boy Advance

公司於21世紀初發售的新一代掌上遊戲機，是GB的第三代掌機。在設計上較其前一代有大幅度改進。在21世紀初可以說是非常流行的機種。

2003
Game Boy Advance SP

Game Boy Advance SP是日本任天堂公司於2003年發售的便攜式遊戲機，是Game Boy Advance的升級產品。

機身為折疊式，顯示屏為帶發光裝置（前光）的32000色反射式TFT液晶顯示屏，解決了Game Boy Advance在黑暗中無法看清屏幕的問題。電池換成了充電鋰電池，充電3小時大約可使用10小時（不開屏幕發光裝置的話可以使用18小時）。本款在硬件的數據處理能力上和GBA沒有甚麼兩樣，但外觀和外圍配置卻發生了很大變化。

2005
Game Boy Micro

能使用GBA遊戲軟體，Game Boy MICRO發售於2005年9月13日，特別為超級瑪利歐20周年紀念。性能等同於Game Boy Advance／Game Boy Advance SP，且非常小巧，便於攜帶。其螢幕缩小至2.0英寸（GBA/SP为2.9英寸），但同時在亮度上大幅度提高並有可調節的背光。

GBM是任天堂采用背光液晶屏幕的Game Boy Advance系列的款式，可做5級的亮度校調。同GBA不可摺疊，但其機型亦為系列中最輕巧的。

Game Boy MICRO也是Game Boy Advance系列中唯一不向下相容以前Game Boy和Game Boy Color遊戲。只支援GBA遊戲，另外連線配件也與Game Boy Advance系列不相容。
